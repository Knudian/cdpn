Different types of broadcast
Unicast
Exemple : une personne A parle à une personne B
Broadcast
Exemple : une personne A parle à tout un groupe de personnes.
Multicast
Exemple : une personne A parle à un groupe de personnes particulier dans une salle
Anycast
Exemple : une personne A parle aux personnes les plus proches d'elles
A natural DDoS repellent

A network is composed of redundant nodes

Response is given by the most efficient node of the group

Can only be broken if the real address of a node is known by black hats

Geocast
Exemples : une personne A parle à un groupe de personnes rassemblées dans une salle précise.

The DomainClient

  • Endpoint for all resources comming from the DomainFactory system
  • Implements the Repository pattern
The DomainManager

  • Uses : Repositories, DomainClient
  • UI for the DomainClient
  • Deep linking in the IS
    => no regression from the past,
    => migrating from old to new system,
    => managing domains by creating subzones and resource records,
    => online editor
The HttpRedirectionManager

  • Uses : Repositories, DomainClient, HttpClient
  • No regression
  • Taking the old and new HTTP systems into account
  • Allowing SSL certificates (HTTPS)
FailOverManager

  • Uses : Repositories, DomainClient, FailOverClient
  • Configuring record checks

FailOver : dynamic record update from checks
